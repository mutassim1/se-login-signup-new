# SE Login Signup Project

This project is part of ICSI 418Y Software Engineering Homework 1.

## Description
This project demonstrates basic front-end web development using HTML and CSS.

It contains two pages:
1. Login Page
2. Signup Page

Both pages use a shared CSS file for consistent styling.

## Files
- `login.html` - Login page
- `signup.html` - Signup page
- `styles.css` - Styling for both pages
